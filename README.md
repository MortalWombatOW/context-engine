# ContextEngine

An MCP server providing workflows that promote human-engaged agentic coding.

> **Status**: Beta. Active development.

## Documentation

The documentation is organized into four sections:

*   [**Tutorials**](docs/tutorials/getting-started.md): Hands-on lessons to get you started.
*   [**How-To Guides**](docs/how-to/installation.md): Solution-focused steps for common tasks.
*   [**Reference**](docs/reference/configuration-schema.md): Technical descriptions and specs.
*   [**Explanation**](docs/explanation/philosophy.md): Background and design philosophy.

## Quick Start

1.  **Install**: `pip install git+https://github.com/MortalWombatOW/context-engine.git`
2.  **Configure**: Create `.context-engine.yaml` in your project.
3.  **Connect**: Add to your MCP client config.

See the [Getting Started Tutorial](docs/tutorials/getting-started.md) for a full walk-through.
